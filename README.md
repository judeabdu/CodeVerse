# 🚀 CodeVerse Academy

CodeVerse Academy is a modern online learning platform designed to teach programming and software development skills through structured courses and hands-on projects.

## ✨ Features

- Modern responsive UI
- PHP-powered backend
- MySQL database integration
- Course catalog system
- Student registration
- Secure login system
- Admin dashboard
- Mobile-friendly design
- Future payment integration

## 📚 Available Courses

- HTML5 & CSS3 Masterclass
- Advanced JavaScript (ES6+)
- Full-Stack PHP & MySQL
- Python Systems & Automation
- AI & Machine Learning
- React & Next.js
- Node.js & Express
- Rust Programming

## 🛠️ Tech Stack

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
